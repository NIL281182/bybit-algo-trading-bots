## 🎯 Какую задачу решает

Трейдинг на криптовалютном рынке требует:
- **Дисциплины** — следовать стратегии, не поддаваясь эмоциям (FOMO, страх).
- **Скорости** — рынок работает 24/7, человек не может мониторить графики круглосуточно.
- **Риск-менеджмента** — точный расчёт размера позиции, стоп-лоссов и плеча.
- **Проверки идей** — перед запуском на реальные деньги стратегия должна быть бэктестирована на исторических данных.

Этот проект решает все эти задачи:
1. **Бэктестер** — проверяет стратегию на истории (2019–2026) с учётом комиссий и проскальзывания.
2. **Торговый бот** — автоматически отслеживает сигналы, входит и выходит по правилам, отправляет уведомления.
3. **Модульная архитектура** — общий код вынесен в `bybit_utils.py`, что позволяет легко добавлять новые стратегии.

- **Начинающие алго-трейдеры** — хотят увидеть, как из теории (стратегия из PDF) рождается рабочий код, и понять, почему субъективные методы не подходят для автоматизации.

- **Практикующие трейдеры** — нужен готовый бот для трендовой торговли BTC/USDT с контролируемой просадкой (~6%) и заранее определёнными правилами входа/выхода.
- **Исследователи** — интересует процесс бэктестирования, фильтрация выбросов в данных и корректный расчёт размера позиции с учётом плеча.

## 🏗 Архитектура

```text
.
├── backtest_donchian_v3.py       # Бэктест Donchian v3 на истории Binance Spot
├── backtest_short_donchian.py    # Бэктест шорт-стратегии (PF < 1.5, отложен)
├── bot_donchian_v3.py            # 🔴 Живой бот BTC/USDT (Bybit Testnet)
├── bot_ema_pullback_eth.py       # 🔴 Живой бот ETH/USDT ( pullback по EMA )
├── bybit_utils.py                # Общий модуль: API, ордера, расчёт qty, логирование
├── alerts.py                     # Telegram-уведомления
├── start_bots.bat / stop_bots.bat # Один клик для запуска/остановки на Windows
├── BOT_SETUP.md                  # Пошаговая инструкция по запуску
├── TELEGRAM_SETUP.md             # Настройка Telegram-алертов
└── fibdiv/                       # Скриншоты и OCR-результаты разбора стратегии FibDiv
```

**Принципы:**
- **DRY** — общий код (`bybit_utils.py`) используется всеми ботами.
- **Безопасность** — API-ключи хранятся в `.env`, файл добавлен в `.gitignore`.
- **Надёжность** — бот использует Testnet по умолчанию; данные для сигналов берутся с Mainnet.
- **Логирование** — каждый бот пишет в свой `.log` файл.

## 📈 Стратегии

### 1. Donchian Breakout v3 (BTC/USDT, Only Long)

| Параметр | Значение |
|----------|----------|
| Таймфрейм | 1D |
| Тип | Тренд-фолловинг, Only Long |
| Вход | `Close > Highest High за 20 дней` И `Close > EMA50` |
| Выход | Свеча закрылась ниже EMA20 (trailing) |
| Стоп-лосс | Entry − 2.0 × ATR(14) |
| Риск на сделку | 1% от equity |
| Плечо | 2× |
| Комиссия (в бэктесте) | 0.06% (вход + выход) |

**Ключевая идея:** торговать в сторону долгосрочного тренда, входить на пробое 20-дневного максимума, выходить при первых признаках замедления (закрытие ниже EMA20). Стоп-лосс защищает от ложных пробоев.

### 2. EMA Pullback (ETH/USDT, Only Long)

Стратегия на откат к EMA на 4H таймфрейме. Бот отслеживает импульсный откат к скользящей средней и входит в направлении тренда.

## 📊 Результаты бэктестов

### Donchian v3 (Binance Spot, 2020-02-20 → 2026-06-07)

| Метрика | Значение |
|---------|----------|
| Сделок | 43 (~7 в год) |
| Win Rate | 44.2% |
| Доходность | **+51.89%** ($10k → $15.2k) |
| Buy & Hold | +551.84% |
| Max Drawdown | **−6.00%** |
| Profit Factor | **2.83** |
| Expectancy | +$120.67 / сделка |

**Вывод:** Стратегия уступает Buy & Hold по абсолютной доходности, но с **в разы меньшей просадкой** (−6% против −77%). Это делает её привлекательной для торговли с плечом ×2–×3 на деривативах, а также для консервативных инвесторов, которым важнее сохранение капитала, чем максимальная доходность.

## ⚡ Быстрый старт

### 1. Клонирование и установка

```bash
git clone https://github.com/NIL281182/bybit-algo-trading-bots.git
cd bybit-algo-trading-bots
pip install -r requirements.txt
```

### 2. Настройка API-ключей (Bybit Testnet)

1. Зарегистрируйтесь на [testnet.bybit.com](https://testnet.bybit.com)
2. Создайте API-ключ (**System-generated**, Read + Write, ❌ Withdraw)
3. Запросите тестовый депозит через Faucet (50 000 USDT)
4. Скопируйте `.env.example` → `.env` и вставьте ключи:

```bash
cp .env.example .env
# Отредактируйте .env:
# BYBIT_API_KEY=your_key
# BYBIT_API_SECRET=your_secret
# TELEGRAM_BOT_TOKEN=... (опционально)
# TELEGRAM_CHAT_ID=... (опционально)
```

### 3. Запуск

```bash
# BTC Donchian Bot
python bot_donchian_v3.py

# ETH Pullback Bot (в другом терминале)
python bot_ema_pullback_eth.py
```

Или на Windows просто дважды кликните **`start_bots.bat`**.

Подробнее — в [`BOT_SETUP.md`](BOT_SETUP.md).

## 🔒 Безопасность

- ✅ **Testnet по умолчанию** — бот торгует фиктивными деньгами.
- ✅ **API-ключи в `.env`** — файл исключён из Git (`.gitignore`).
- ✅ **Withdraw отключён** — у API-ключа нет права на вывод средств.
- ✅ **Правильный расчёт qty** — плечо **не увеличивает** размер позиции, а только снижает требуемый margin. Это предотвращает перерасход капитала.
- ✅ **Фильтр выбросов** — при загрузке свечей удаляются аномальные значения (биржевые баги).
